# World Clock

🇬🇧 [English](README.md)

Виджет для рабочего стола Windows, который одновременно показывает текущее время в нескольких городах мира в разных часовых поясах — прямо над панелью задач.

## Возможности

- Отображение времени в нескольких часовых поясах одновременно
- Поиск и добавление любых городов мира
- Метки городов переключаются между кириллицей и латиницей при смене языка
- Всегда поверх других окон, не мешает работе
- Иконка в системном трее для быстрого доступа
- Интерфейс на русском и английском языках

## Установка

1. Скачайте файл `World Clock Setup 1.0.0.exe` на странице [Releases](../../releases)
2. Запустите установщик двойным кликом
3. В появившемся окне UAC нажмите **Да** — приложение требует права администратора для отображения поверх панели задач
4. Установка завершится автоматически, на рабочем столе появится ярлык

## Использование

После запуска часы появятся в нижнем левом углу экрана над панелью задач.

**Иконка в трее** (правый нижний угол экрана):
- **Правый клик** — меню: показать/скрыть, настройки, выход
- **Двойной клик** — открыть настройки

**Настройки:**
- Поиск города по названию (на русском или английском)
- Клик по городу в списке — добавить
- Кнопка `×` напротив города — удалить
- Переключатель языка внизу окна настроек

## Язык интерфейса

По умолчанию приложение запускается на **английском**. Чтобы переключиться на русский, откройте Настройки и нажмите **RU — Русский** внизу окна. Все подписи — включая коды городов в виджете — обновятся мгновенно.

## Автозапуск с Windows

Чтобы часы запускались автоматически при входе в систему:

1. Нажмите `Win + R`, введите `shell:startup`, нажмите **Enter**
2. Скопируйте ярлык **World Clock** с рабочего стола в открывшуюся папку

Теперь часы будут запускаться вместе с Windows.

## Закрепление иконки в трее

Приложение работает как виджет — окно часов всегда видно поверх других программ. Управление осуществляется через иконку в **системном трее** (область уведомлений рядом с часами Windows).

Чтобы иконка трея всегда была видна:

1. Правый клик на панели задач → **Параметры панели задач**
2. **Область уведомлений** → **Выбор значков**
3. Найдите **World Clock** и включите отображение
